# TTFN RSVP Site

Entry points:
- `index.html` redirects to `rsvp.html` (main live page).
- `rsvp-app.html` is the API-backed RSVP flow.
- `edit.html` handles attendee edits via `editUrl`.
- `admin.html` is the RSVP dashboard.

Static assets live in `images/`. Legacy materials are stored in `archive/`.
